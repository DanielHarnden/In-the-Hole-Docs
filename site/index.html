This is documentation for the hit 2026 video game In the Hole.

Using MkDocs

For full MkDocs documentation visit mkdocs.org.

Some quick commands (execute them in the command prompt in the Documentation/ folder) for MkDocs:

  • mkdocs serve - Start the live-reloading docs server.
  • mkdocs build - Build the documentation site.
  • mkdocs -h - Print help message and exit.

To access the documentation without having the project open, open the HTML file Documentation/site/index.html.
